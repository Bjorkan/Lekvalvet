
# Lekvalvet 🪁

Lekvalvet är ett arkiv av många lekar som man kan leka för att skapa en bra gruppdynamik. Sidan växer hela tiden med nya övningar och ska förhoppningsvis kunna hjälpa folk som leder grupper. Drivs med Material for MkDocs och är hostad av GitHub Pages.
